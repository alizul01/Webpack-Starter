<h2 align="center">Install</h2>

Install with npm:

```bash
npm install --save-dev webpack
```

Install with yarn:

```bash
yarn add webpack --dev
```

<h2 align="center"> How to use this starter </h2>

Run Development Mode
```bash
npm run dev
```

Run Build Mode
```bash
npm run build
```
